# DES
Implementation of the Data Encryption Standard algorithm using C++

## Getting Started
- setup g++
for ubuntu 20.04 LTS
```
sudo apt-get install g++
```
- compile
```
g++ DES.c -o DES
```
- run
```
./DES <encrypt/decrypt> <plaintext/cipher> <key>
```
all in hexadecimal form
